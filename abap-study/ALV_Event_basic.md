# ABAP ALV 이벤트 기본 정리

## 1. ALV 이벤트란?
- ALV 화면에서 발생하는 사용자 동작을 처리하는 기능
- 더블클릭, 버튼 클릭, 데이터 변경 등에 반응

---

## 2. 자주 쓰는 이벤트

### Double Click
- 특정 행을 더블클릭했을 때 발생
- 상세 화면 이동에 주로 사용

### Hotspot Click
- 특정 컬럼을 링크처럼 클릭
- 문서번호 클릭 → PO 조회 등

### User Command
- 툴바 버튼 클릭 시 발생
- 사용자 정의 기능 처리

---

## 3. 이벤트 처리 흐름
1) 이벤트 핸들러 클래스 생성
2) 이벤트 메서드 정의
3) ALV Grid에 이벤트 등록
4) 이벤트 발생 시 메서드 실행

---

## 4. 실무 활용 예
- PO 번호 더블클릭 → ME23N 이동
- 특정 컬럼 클릭 → 상세 데이터 조회
- 사용자 버튼으로 엑셀 다운로드 실행

---

## 5. 컨설턴트 관점 포인트
- “조회만 하는 ALV” vs “인터랙션 있는 ALV” 구분
- 사용자 요구사항 대부분이 이벤트 처리로 해결
- ALV 이벤트 구조를 알면 개발자와 소통 쉬움
