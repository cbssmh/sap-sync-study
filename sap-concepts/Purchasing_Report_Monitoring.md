# SAP MM 구매 리포트 & 모니터링 정리

## 1. 구매 리포트 & 모니터링이란?
- 구매 프로세스 전반의 상태를 조회·관리하는 기능
- 진행 중인 문서, 지연, 오류를 한눈에 파악

---

## 2. 모니터링의 목적
- 구매 진행 현황 파악
- 지연/미처리 문서 조기 발견
- 업무 병목 구간 확인

---

## 3. 주요 구매 리포트 T-Code

### ME2N / ME2L / ME2K
- PO 리스트 조회
- 공급업체/구매조직/계약 기준 조회

### ME5A
- PR 리스트 조회
- 승인 대기 PR 확인

### ME80FN
- 구매 문서 종합 조회
- PR/PO/계약 통합 모니터링

---

## 4. 입고/송장 모니터링

### MB5S
- 미입고 PO 조회
- GR 지연 확인

### MRBR
- 송장 블로킹 리스트
- 지급 불가 사유 확인

---

## 5. 자주 보는 체크 포인트
- 승인 대기 PR
- 미입고 PO
- GR/IR 잔액
- 송장 블로킹 상태

---

## 6. 실무 포인트
- 문제는 대부분 리포트에서 먼저 발견됨
- 정기 모니터링으로 이슈 예방 가능
- 컨설턴트는 “어디서 막혔는지” 리포트로 설명 가능해야 함
