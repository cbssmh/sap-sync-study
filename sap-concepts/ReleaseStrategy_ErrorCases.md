# SAP MM Release Strategy 오류 케이스 정리

## 1. 승인(Release)이 안 걸리는 경우
- 금액 조건 미충족
- 문서 유형(Document Type) 불일치
- Release Strategy 조건 누락

실무 포인트:
- 실제 금액 vs 조건 금액 먼저 비교

---

## 2. 승인 대상인데 승인 버튼이 안 보이는 경우
- Release Code에 사용자 미할당
- 사용자 Role/권한 부족

확인 사항:
- 승인자 매핑
- 권한 오브젝트

---

## 3. 승인 후 변경 시 다시 승인 안 되는 경우
- Changeability Indicator 설정 문제
- 변경 허용 필드로 인식됨

실무 포인트:
- 가격/수량 변경 시 재승인 필요 여부 확인

---

## 4. 승인 완료 후 PO 생성 불가
- Release Indicator 설정 오류
- 승인 완료 상태로 전환되지 않음

확인 사항:
- 승인 완료 상태 값
- 후속 문서 허용 설정

---

## 5. 승인 전략이 전혀 적용 안 되는 경우
- Classification 특성 값 누락
- 전략과 문서 연결 실패

실무 포인트:
- 조건 클래스/특성 값부터 점검

---

## 6. 실무 정리
- 승인 오류의 대부분은 설정/조건 문제
- “금액 → 조건 → 승인자 → 상태” 순서로 점검
- 컨설턴트는 오류 재현 + 원인 설명이 가능해야 함
