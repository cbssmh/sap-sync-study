# SAP MM 재고 관리 (Inventory Management) 정리

## 1. Inventory Management란?
- 자재의 입고, 출고, 이동, 재고 수량을 관리하는 영역
- MM 모듈의 핵심 기능 중 하나
- 재고 변동 시 회계(FI)와 자동 연동됨

---

## 2. 재고 관리의 주요 기능
- 입고(Goods Receipt)
- 출고(Goods Issue)
- 재고 이동(Transfer)
- 재고 수량 및 가치 관리

---

## 3. 재고 관리의 기준 단위
- Plant : 재고 관리의 기본 단위
- Storage Location : 실제 보관 위치
- 자재(Material) 단위로 재고 관리

---

## 4. 주요 재고 처리 유형
- PO 기반 입고 (101)
- 입고 취소 (102)
- 생산 출고 (261)
- 비용 출고 (201)
- 재고 이동 (311 등)

---

## 5. 재고 관리와 회계 연계
- 재고 증감 시 회계 전표 자동 생성
- 자산 계정, 비용 계정에 즉시 반영
- 잘못된 재고 처리 = 금액 오류로 직결

---

## 6. 실무 포인트
- 재고 오류 문의는 Movement Type부터 확인
- Plant/Storage Location 설정 오류 빈번
- 수량 문제인지, 금액 문제인지 먼저 구분
- MM 컨설턴트는 재고 흐름을 설명할 수 있어야 함
