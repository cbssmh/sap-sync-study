# SAP MM 재고 실사 (Physical Inventory) 정리

## 1. 재고 실사란?
- 시스템 상 재고와 실제 재고를 비교·조정하는 프로세스
- 정기적/수시로 수행하여 재고 정확도 확보

---

## 2. 재고 실사 목적
- 재고 차이(분실, 파손, 오입력) 확인
- 재무 신뢰성 확보
- 감사 및 내부 통제 대응

---

## 3. 재고 실사 기본 흐름
1) 실사 문서 생성
2) 실제 재고 수량 카운트
3) 차이 입력
4) 차이 전기(조정)
5) 재고/회계 반영

---

## 4. 주요 T-Code
- MI01 : 실사 문서 생성
- MI04 : 실사 수량 입력
- MI07 : 차이 전기
- MI20 : 실사 리스트 조회

---

## 5. 실사 유형
- 연간 실사 (Year-end)
- 순환 실사 (Cycle Counting)
- 임의 실사 (Ad-hoc)

---

## 6. 실무 포인트
- 실사 중에는 재고 이동 제한
- 차이 전기 시 회계 전표 자동 생성
- 실사 오류의 대부분은 Storage Location 선택 실수
- 컨설턴트는 “차이 발생 원인”을 설명할 수 있어야 함
