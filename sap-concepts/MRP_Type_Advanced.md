# SAP MM MRP 유형 심화 정리

## 1. MRP 유형이란?
- 자재별로 소요를 어떻게 계산할지 결정하는 설정
- 자재 마스터 MRP View에서 지정
- MRP 실행 결과에 직접적인 영향

---

## 2. 주요 MRP 유형 상세

### PD – 표준 MRP
- 실제 소요량 기반으로 계산
- 생산/구매 제안 자동 생성
- 가장 일반적으로 사용

---

### VB – 재주문점 기반
- 재고가 재주문점 이하로 내려가면 조달
- 소요 예측보다 단순 관리에 적합
- 소모성 자재에 자주 사용

---

### ND – MRP 미실행
- MRP 계산 대상 제외
- 수동 구매 자재에 사용

---

### V1 – 자동 재주문점
- 과거 소요 데이터를 기반으로 재주문점 자동 계산
- 시스템 주도 관리

---

### VV – 수동 재주문점
- 사용자가 재주문점 직접 설정
- 단순하지만 관리 부담 있음

---

## 3. MRP 유형 선택 기준
- 수요 변동 큼 → PD
- 소모성/저가 자재 → VB, VV
- 계획 불필요 자재 → ND

---

## 4. 실무 포인트
- 잘못된 MRP 유형 설정 = 불필요한 PR 생성
- 재고 과잉/부족의 주요 원인
- 자재 성격에 맞는 유형 선택 중요
- 컨설턴트는 “왜 이 MRP 유형을 쓰는지” 설명 가능해야 함
